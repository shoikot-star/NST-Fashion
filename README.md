# NST Fashion 👗

An elegant e-commerce platform built with **Python Django** and **Tailwind CSS**.

## ✨ Features
- **Product Management:** Dynamic product listing and detail pages.
- **Shopping Cart:** Add to cart and manage items.
- **User Profile:** Dedicated profile page with shipping address management.
- **Secure Checkout:** Integrated order processing system.
- **Responsive Design:** Premium UI using Tailwind CSS.

## 🛠️ Tech Stack
- **Backend:** Django 6.0+
- **Frontend:** Tailwind CSS, HTML5
- **Database:** SQLite (Development)

## 🚀 How to Run Locally
1. Clone the repo: `git clone https://github.com/shoikot-star/NST-Fashion.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run migrations: `python manage.py migrate`
4. Start server: `python manage.py runserver`
